# 演練隊長小幫手

協助小隊長快速進行演練配對與會議安排的專業工具。

## 功能特色

### 核心功能
- ✅ 雙層帳戶系統（大隊長 + 小隊長）
- ✅ 組員資料管理（天賦、初訓/複訓、職業、備考）
- ✅ 8種天賦系統（羅傑漢彌頓原動力系統）
- ✅ 黃金三角配對優化
- ✅ 一鍵自動排序
- ✅ 表格式手動微調編輯
- ✅ 配對結果自動連動更新
- ✅ 模板複製與自訂
- ✅ 會議三角配置
- ✅ LINE 接龍模板生成

### 配對優先級（從高到低）
1. **避免重複配對** ⭐⭐⭐⭐⭐ - 除非所有人都輪過，否則絕不重複
2. **複訓配初訓** ⭐⭐⭐⭐ - 盡量每個初訓生都配複訓生
3. **平等考慮** ⭐⭐⭐
   - 黃金三角組合
   - 相同或相似天賦
   - 相同職業

## 快速開始

### 環境要求
- Node.js 18+
- Supabase 帳戶
- npm 或 yarn

### 安裝步驟

1. **克隆項目**
```bash
cd /Users/thewolf/自己做的酷東東/016演練小幫手
```

2. **安裝依賴**
```bash
npm install
```

3. **設置數據庫**
   - 查看 [DATABASE_SETUP.md](./DATABASE_SETUP.md) 了解詳細步驟
   - 在 Supabase 中運行 `lib/database.sql`

4. **配置環境變數**
```bash
cp .env.example .env.local
# 編輯 .env.local，添加 Supabase 認證信息
```

5. **運行開發服務器**
```bash
npm run dev
```

6. **訪問應用**
   - 打開 http://localhost:3000
   - 首次使用需要在 Supabase 中創建帳戶

## 配對演算法說明

### 黃金三角規則
黃金三角由三個天賦組成：
- 某天賦的 +2 天賦
- 該天賦的 +3 天賦

**例子：地主的黃金三角**
- 地主 (+2) → 創作者 (+3) → 媒合者
- **黃金三角組合：地主、創作者、媒合者**

### 配對流程
1. 檢查未配對過的人員組合（優先級1）
2. 優先複訓配初訓（優先級2）
3. 評估黃金三角、相似天賦、職業互補（優先級3）
4. 生成最優配對方案

## 使用指南

### 大隊長操作
1. 登入大隊長帳戶
2. 設定班期信息（開課日、結業日、總週數）
3. 編輯公版模板
4. 查看平台概況

### 小隊長操作
1. 登入小隊長帳戶
2. 新增組員信息
3. 設定會議時間
4. 執行演練配對排序
5. 複製模板到 LINE 群

## 常見問題

**Q: 如何重置密碼？**
A: 在 Supabase 儀表板中管理用戶認證。

**Q: 如何備份數據？**
A: Supabase 提供自動備份功能，可在儀表板中配置。

**Q: 可以支持多個班期嗎？**
A: 是的，系統支持多個班期和戰隊。
